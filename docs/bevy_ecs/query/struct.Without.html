Struct Without 

Source 
pub struct Without<T>(/* private fields */);
Expand description

Filter that selects entities without a component T.

This is the negation of With.

§Examples

fn no_permit_system(query: Query<&Name, Without<Permit>>) {
    for name in &query{
        println!("{} has no permit!", name.name);
    }
}
