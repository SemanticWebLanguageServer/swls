Expand description

Very similar to a normal Column, but with the capacities and lengths cut out for performance reasons.

This type is used by Table, because all of the capacities and lengths of the Table’s columns must match.

Like many other low-level storage types, ThinColumn has a limited and highly unsafe interface. It’s highly advised to use higher level types and their safe abstractions instead of working directly with ThinColumn.
