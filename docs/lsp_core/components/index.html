Defines all common [Component]s and [Resource]s

In this [World], [Entity]s are documents and Components are derived from these documents. Different [System]s derive new Components from existing Components, that are added to the [Entity]. For example, if Triples are defined, systems::derive_classes will derive DefinedClass from them and add them to the [Entity].
