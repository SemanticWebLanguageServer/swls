Struct MakeWebConsoleWriter

pub struct MakeWebConsoleWriter { /* private fields */ }
Expand description

A MakeWriter emitting the written text to the console.

The used log method is sensitive to the level the event is emitted with.

LevelMethod
TRACEconsole.debug
DEBUGconsole.debug
INFOconsole.info
WARNconsole.warn
ERRORconsole.error
otherconsole.log
